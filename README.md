## SegmentDigit
A library to display digits on any display using xy coordinates and with a function to draw rectangles

Currently supported:
Integers from 1-4 digits
Float 4 digits
Time 4 digits ( HH:MM or MM:SS )
Time 6 digits (HH:MM:SS )
